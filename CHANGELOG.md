# Changelog

## 0.2.0 - 2026-06-27

- Added superadmin pipeline filters for agent name, highest-match product, date range, budget, age, and status.
- Added agent-name labels on superadmin contact cards while keeping agent views scoped and uncluttered.
- Resolved lead agent labels from `agent_slug` through agent records when the saved lead lacks `agent_name`.
- Converted agent management to a searchable table and added agent detail tabs for leads and settings.
- Preserved `agent_name` on authenticated agent lead saves so D1 lead records stay attributable.
- Allowed agents to revisit review options and update an existing quotation before resending.
- Expanded the workspace to fluid screen width and changed the primary accent to dark red.
